# Hogyan érkeztél? - Interaktív Szavazás

Ez egy egyszerű, valós idejű szavazóalkalmazás, amely arra szolgál, hogy egy csoport (például egy osztály) gyorsan és vizuálisan felmérje, ki milyen közlekedési eszközzel érkezett.

## A program célja

A program célja, hogy interaktív és látványos módon gyűjtsön adatokat a közlekedési szokásokról. Kiválóan használható tanórai felmérésre, például a fenntarthatósági témahét vagy egy közlekedésről szóló téma keretében.

## Hogyan működik?

1.  **Szavazás:** Kattints arra az ikonra, amelyik a te utazási módodat jelképezi: autó, tömegközlekedés, bicikli vagy gyalog.
2.  **Azonnali visszajelzés:** Minden kattintás után az oszlopdiagram azonnal frissül, mutatva a szavazatok aktuális állását.
3.  **Adatok letöltése:** Ha szeretnéd elmenteni a szavazás végeredményét, nyomd le az **Alt + Ctrl + T** billentyűkombinációt. A program ekkor egy CSV formátumú fájlt generál és tölt le az adatokkal, amelyet táblázatkezelőben (pl. Excel, Google Sheets) megnyithatsz.

## Kinek ajánlott?

* **Tanároknak:** Gyors és játékos tanórai adatgyűjtésre, ami azonnal vizualizálja az eredményeket.
* **Diákoknak:** Hogy részt vegyenek egy egyszerű adatgyűjtési folyamatban, és lássák, hogyan alakul egy statisztika.
